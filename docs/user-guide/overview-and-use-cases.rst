Overview and Use Cases
======================

OpenBook is the successor to lecture-slide.js and aims to provide interactive
online textbooks for teaching and learning. The long-term goal is not only to
publish static learning material, but to support adaptive learning workflows,
authored and maintained in a modern web application.

At this stage, the project is still under construction. Core backend and
authentication workflows are already available, while larger user-facing
features, such as complete authoring workflows, are being built incrementally.

Current problem focus
---------------------

The project addresses three recurring problems in everyday teaching:

1. Linear material does not fit all learners equally well.
2. Teachers lack transparent feedback about student learning progress.
3. Students lose orientation about priorities, timelines, and prerequisites.

Near-term direction
-------------------

The active roadmap focuses on practical building blocks:

- Better content authoring support.
- Richer interactive elements and third-party integrations.
- More structured textbook and catalogue workflows.
- Improved feedback and account workflows.

You can follow technical roadmap details in the developer and maintainer
manual pages.
