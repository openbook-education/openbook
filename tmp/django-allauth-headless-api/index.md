Django Allauth Flows
====================

The files in this directory document the screen flows used in our application, that are
implemented via Django Allauth. The purpose is to document the screens that need to be
implemented in the frontend SPA as well as the required backend API calls. Beyond initial
implementation this serves as a living specification to document how authentication works
from the frontend's or other clients point of view.

To keep the documentation maintainable, it has been split into several files.

1. [Basic API Principles](api-principles.md)
1. [Login and Logout](basic-login-logout.md)
1. [Social Login with SAML](social-login-saml.md)
1. [Account Management](account-management.md)
