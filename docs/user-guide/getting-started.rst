Getting Started
===============

OpenBook currently has two practical entry points: trying the running platform
and working with account flows used by the frontend and admin interfaces.

If you are looking for local setup or deployment instructions, use the
Developer and Maintainer Manual. This user-facing chapter intentionally stays
lightweight until more end-user workflows are available.

What you can do today
---------------------

You can already explore and test:

- Login and logout with local accounts.
- Account signup and email verification.
- Password reset and session status checks.
- Social login and SAML-based identity provider flows.

Where to go next
----------------

Continue with :doc:`/user-guide/accounts-and-authentication` for the currently
stable user-level authentication flows.
